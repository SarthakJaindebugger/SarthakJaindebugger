A privacy-aware full-stack AI system designed for immigrant support workflows with the City of Espoo context at Aalto University.

**Tech focus:** multilingual ASR/TTS, GenAI orchestration, speaker diarization, RAG, vector database retrieval, BM25 keyword search, React, Flask, Docker.

**Engineering highlights**

- Designed an orchestration layer for text and speech modalities.
- Combined semantic retrieval with BM25 for robust query answering.
- Planned user/admin interfaces with privacy-aware data handling.
- Containerized the web application for reproducible deployment.
